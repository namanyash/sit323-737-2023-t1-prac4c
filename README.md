# sit323-737-2023-t1-prac4p

sit323/737-2023-t1-prac4p

Accepts following API endpoints
`getToken`
returns a valid JWT token

`/api/add`
Requires header (`x-auth-token`) with a valid token
Requires two variables `a` and `b`
returns `a+b`

`/api/subtract`
Requires header (`x-auth-token`) with a valid token
Requires two variables `a` and `b`
returns `a-b`

`/api/divide`
Requires header (`x-auth-token`) with a valid token
Requires two variables `a` and `b`
returns `a/b`

`/api/multiply`
Requires header (`x-auth-token`) with a valid token
Requires two variables `a` and `b`
returns `a*b`
